Brief About the Project

Building a Simple Spam Detection System using Machine Learning to train a Logistic Regression Model by feeding the Model with a dataframe consisting of both spam and harmless content in order to make future predictions.
Django framework is used to manage both the frontend (i.e html, css etc) and the backend (i.e python) and the text or file uploaded is run through the Model which prints the prediction and evaluation reports.

Note: The Accuracy of the Model is based on the Dataset used in the training Stage Credits: Dataset Source - Siddhardhan
